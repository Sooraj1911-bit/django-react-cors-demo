# Django React CORS Demo

This project demonstrates how to connect a Django backend with a React frontend using CORS.

---

## Tech Stack
- Django
- Django REST Framework
- django-cors-headers
- Git & GitHub

---

## How to Run Project

1. Clone the repository
git clone https://github.com/Sooraj1911-bit/django-react-cors-demo.git

2. Go to project folder
cd django-react-cors-demo

3. Install dependencies
pip install django djangorestframework django-cors-headers

4. Run server
python manage.py runserver

Open browser:
http://127.0.0.1:8000/
